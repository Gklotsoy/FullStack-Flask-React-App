# Diary Full Stack Application

## Overview
This project is a full-stack diary application that allows users to create, read, update, and delete diary entries. The application is built using modern web technologies and follows best practices for both frontend and backend development.

## Features
- User authentication and authorization
- Create, read, update, and delete diary entries

## Technologies Used
### Frontend
- HTML
- CSS
- JavaScript
- React

### Backend
- Python
- Flask

## Database
-Mysql

## Installation
1. Clone the repository:
    ```
    git clone https://github.com/yourusername/diary-full-stack.git
    ```
2. Create MySql Database
    ```
    Database name: dbfullstackapp (or set the database name you want in the conig.py file)

    Notice: if you have a password set for your database make sure to make the necessary adjastments in the conig.py file
    ```
3. Navigate to the project directory:
    ```
    cd diary-full-stack
    ```
4. Install dependencies for both frontend and backend:
    ```
    npm install
    cd client
    npm install
    cd ..
    ```

## Usage
1. Start the backend server:
    ```
    npm run server
    ```
2. Start the frontend development server:
    ```
    cd client
    npm start
    ```
